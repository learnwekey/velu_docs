Velu is an AI-native documentation platform for creating, publishing, and
maintaining documentation. Teams can write in the web editor or manage MDX
content in Git with Velu's command-line tools.

Velu combines authoring, reusable components, API documentation, and
AI-native discovery in one documentation experience. Use the paths below to
start publishing or explore a specific part of the platform.
